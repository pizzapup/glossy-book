# Typography + Color Guidelines

## Typography

Choose fonts carefully! A single font file could be as much as 250kb, and that might only be for the standard weight.

- **Use system fonts where possible**
- **Use less font variations**
- Technical strategies can be used to minimise the size of fonts on websites and can result in file size reductions of up to 97%. Try these techniques to reduce the size of font files:
- Stick to modern web font file formats like WOFF and WOFF2, which use higher compression methods compared to TTF, OFT and SVG file formats.
- Subset fonts to only include the characters needed on the website.
  For more information about optimising performance and size of web fonts, [check out this article by Joshua Stopper](https://www.wholegraindigital.com/blog/performant-web-fonts/)

## Color

### Classifying Primary, Secondary, and Tertiary Actions

- **Primary actions** should be obvious. Solid, high contrast background colors work great here.
- **Secondary actions** should be clear but not prominent. Outline styles or lower contrast background colors are great options.
- **Tertiary actions** should be discoverable but unobtrusive. Styling these actions like links is usually the best approach.

**Web Content Accessibility Guidelines (W3C) [Use of Color](https://www.w3.org/TR/UNDERSTANDING-WCAG20/visual-audio-contrast-without-color.html)**
Color is not used as the only visual means of conveying information, indicating an action, prompting a response, or distinguishing a visual element. (Level A)
[Understanding Success Criterion 1.4.1](https://www.w3.org/TR/2008/REC-WCAG20-20081211/#visual-audio-contrast-without-color)

1. Use color and weight to create hierarchy instead of size
2. Reduce contrast when working with colorful backgrounds by reducing the opacity of white text, or hand-picking a color based on the background color (choose a color that’s the same hue as the background, adjusting saturation and lightness) <br/>

   - Reduce opacity: `color: hsl(0, 0%, 90%)` --> `color: hsl(0, 0%, 100%, 0.86)`<br />
   - Choose a color that’s the same hue as the background and adjust the saturation
     and lightness `color: hsl(176, 100%, 84%)` `background: hsl(176, 100%, 24%)`

     >

3. Add vertical offset to shadows instead of large blur and spread values. `box-shadow: 0 4 6px 0 hsla(0, 0%, 0%, 0.2)` `box-shadow: inset 0 2px 4px 0 hsla(0, 0%, 0%, 0.08)`

4. Border alternatives: box-shadow, different background colors, extra spacing. `box-shadow: 0 2px 6px 0 hsla(0, 0%, 0%, 0.2)` / `margin-bottom: 48px`
